# rumour-map

A Vue.js 3 application with Vite and Tailwind CSS for viewing high-resolution maps with pan and zoom capabilities.

## Features

- **Vue.js 3** with Composition API
- **Vite** for fast development and building
- **Tailwind CSS** for utility-first styling
- **GitHub Primer CSS** (spec-kit) for GitHub-style UI components
- **Pan and Zoom** functionality for high-resolution images (6500 x 3600)
- Touch support for mobile devices
- Responsive design

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Usage

The application displays a high-resolution map that can be:
- **Panned** by clicking and dragging with the mouse or touch
- **Zoomed** using the mouse wheel or pinch gestures on touch devices
- **Controlled** using the zoom buttons in the top-right corner
- **Reset** to fit the screen using the reset button
